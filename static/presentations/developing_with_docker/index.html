Future: cgroups

Control cgroups, usually referred to as cgroups, are a Linux kernel feature which which allow processes to be organized into hierarchical groups whose usage of various types of resources can then be limited and monitored. The kernel's cgroup interface is provided through a pseudo-filesystem called cgroupfs - http://man7.org/linux/man-pages/man7/cgroups.7.html

  • cgroup is a collection of processed bound to a set of limits defined via cgroup filesystem
  • Resource controllers - CPU time partition and accounting - Freezing / resuming execution
  • cgroups v2 unified hierarchy
  • single hierarchy rather than multiple mount points allowing unified, atomic behavior
  • systemd organizes processes with cgroups
